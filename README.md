# Secador Solar

## Descripción
El deshidratado solar es una herramiennta que aprovecha los recursos naturales para conservar los alimentos y reducir el desperdicio.
Este proyecto tiene como objetivo el desarrollo de un sistema para calcular el tiempo de secado y peso final del producto deshidratado, orientado a la optimización del proceso de secado de productos agrícolas y alimenticios utilizando energía solar.Tambien se planea que se autilizado a la hora de diseñar el deshidratador, para tener datos teorico y saber si realmente funcionara de una manera corresta y si cumplira las necesidades del usuario. El sistema busca ser eficiente, sostenible y de bajo costo, aprovechando recursos renovables para reducir el impacto ambiental y mejorar la calidad del secado.
## Características principales
- Implementación de algoritmos para el control y monitoreo del proceso de secado.
- Automatización de parámetros como temperatura, humedad y flujo de aire.
- Interfaz para visualización y registro de datos.
- Diseño modular y escalable.
- Analissi de los resultados.

## Estructura del proyecto
- `main.py`: Archivo principal de ejecución del sistema.
- `Codrigo_del_secador.py`: Módulo con la lógica de control del secador solar.
- `requirements.txt`: Lista de dependencias necesarias para ejecutar el proyecto.
- `secador 1/`: Carpeta con recursos adicionales, módulos o datos relacionados.

## Instalación
1. Clona el repositorio:
	```bash
	git clone https://github.com/ReneRodriguez-ing/secador_solar.git
	```
2. Accede al directorio del proyecto:
	```bash
	cd secador_solar
	```
3. Instala las dependencias:
	```bash
	pip install -r requirements.txt
	```

## Uso
Ejecuta el archivo principal para iniciar el sistema:
```bash
python main.py
```
